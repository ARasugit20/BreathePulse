# Changelog

All notable changes to BreathePulse are documented here.

## [1.1.0] - 2026-06-06

### Added
- Session history page with pattern filtering and delete support
- Dark mode with persisted theme preference
- Keyboard shortcuts for breathing sessions (Space, P, Esc)
- Post-session notes dialog
- HRV trend line chart on dashboard
- Daily goal progress widget
- Deployment configs for Vercel (frontend) and Fly.io (backend)

## [1.0.0] - 2026-06-06

### Added
- Full-stack monorepo rebuild (FastAPI + React + PostgreSQL)
- Guided breathing engine (Box, 4-7-8, Wim Hof)
- Session tracking and analytics dashboard
- Pulse calibration page
- User profiles with Supabase JWT auth
- pytest, Vitest, and Playwright test suites
- GitHub Actions CI pipeline
- Docker Compose for local development
